#Recruitment App

##Duration:

7 days

##Tech Stack:
- HTML
- CSS
- JavaScript
- Ruby
- Rails
- Heroku

##Features:
- Browse the site as a guest.
- Users can create, edit and delete accounts.
- Users can login.
- Logged in users can post, edit and delete candidate and client - profiles, job ads and applications.

##Nice to have:
Below are features I would like to implement in the future:

- Responsive design.
- Nested routes for users.
- Reviews component
- Ability to collect more data which is relevant to labour hire and recruitment.
- Email confirmation for new accounts.
- Contact page.
